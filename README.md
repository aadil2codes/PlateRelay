# PlateRelay

**PlateRelay** is a web-based platform designed to reduce food waste by connecting surplus food providers with NGOs and other verified helping organizations in real time.

Every day, large quantities of edible food are wasted by restaurants, hotels, college messes, and events. PlateRelay helps bridge this gap by enabling food providers to quickly post surplus food and allowing NGOs or helping organizations to accept and collect it before it goes to waste.

## Features

* User Authentication using Firebase
* Surplus Food Posting System
* Real-Time Food Listings
* NGO Dashboard
* Accepted Pickups Management
* Pickup Confirmation System
* Google Maps Integration
* Contact & Coordination Details
* Real-Time Database Updates using Firestore

## Tech Stack

### Backend & Services

## How It Works

1. Food providers post surplus food details.
2. NGOs and verified helping organizations view available food.
3. Organizations accept pickup requests.
4. Food is collected from the provider's location.
5. Pickup is confirmed and the listing is updated automatically.

## Project Vision

PlateRelay aims to create a reliable food redistribution network where surplus food reaches people in need instead of being wasted. Our goal is to reduce food waste, support communities, and promote sustainable food management through technology.
